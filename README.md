# Adsum Pater Web 🌿

Sitio web oficial de la misión universitaria **Adsum Pater**, una iniciativa de la Juventud de Schoenstatt que se desarrolla durante Semana Santa, donde jóvenes salen a misionar llevando un mensaje de fe, esperanza y servicio a distintas comunidades :contentReference[oaicite:0]{index=0}.

## 🚀 Sobre el proyecto

Esta plataforma fue desarrollada con el objetivo de centralizar la información de la misión y ofrecer una experiencia digital moderna para:

- Difundir la identidad y espíritu de la misión
- Presentar el destino anual (pueblo misionado)
- Compartir reflexiones, contenido espiritual y materiales
- Gestionar intenciones y participación de la comunidad
- Mostrar cronogramas, historia y equipo organizador

## 🛠️ Tecnologías utilizadas

- **.NET 8 - Blazor WebAssembly**
- **Firebase Realtime Database**
- **Firebase Hosting**
- **HTML / CSS / JS**
- **Integraciones con JS (PDF viewer, UI interacciones)**

## ✨ Funcionalidades principales

- 🔥 Página dinámica de misión anual (ej: Adsum 2026)
- ⏳ Cuenta regresiva hasta la misión
- 🙏 Sistema de intenciones con interacción de usuarios
- 📖 Sección de reflexiones con visualización de PDFs
- 🗺️ Información del pueblo y mapa embebido
- 📅 Cronograma de actividades
- 🧑‍💻 Panel de administración personalizado
- 📦 Gestión de contenido desde Firebase

Brindar una herramienta digital que acompañe la misión, fortalezca la comunidad y acerque la experiencia de Adsum Pater a más personas, incluso fuera del territorio misionado.
